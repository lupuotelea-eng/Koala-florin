Wild Koala Quest - Phaser.js Pro Starter

Conținut:
- index.html
- main.js

Cum îl publici pe GitHub Pages:
1. Creezi un repository public.
2. Încarci ambele fișiere în rădăcina repository-ului.
3. Settings > Pages > Deploy from a branch.
4. Branch: main, Folder: /(root).
5. Aștepți apariția linkului public.

Important:
- index.html trebuie să rămână cu acest nume.
- main.js trebuie să stea lângă index.html.
- Jocul încarcă Phaser din CDN, deci la prima deschidere este nevoie de internet.

Controale:
- Mișcare: WASD sau săgeți
- Interacțiune / continuare dialog: SPACE
